❓ Ayuda – Sol & Verde

Guía rápida de uso del sistema. Podés editar estos textos desde Configuración › Textos / Ayuda.

📊 Dashboard

Muestra el resumen del día, saldos de clientes y accesos rápidos a las operaciones más frecuentes.

👤 Visor de Cliente

Buscá un cliente por nombre para ver su historial de movimientos, saldo actual y datos fiscales.

💸 Nuevo Movimiento

Registrá un FIADO (DEBE) o un PAGO (HABER) para un cliente. El saldo se actualiza automáticamente.

🧾 Facturación ARCA

Emitís comprobantes electrónicos (Factura A/B) directamente desde ARCA. Requiere configurar el Access Token en Configuración › Facturación ARCA.

🏠 Alquileres

Gestioná los pagos semanales y facturas mensuales de cada inquilino. El calendario muestra el estado de cada día.

📒 Caja Diaria

Abrí y cerrá la caja del día, registrá movimientos manuales y realizá el arqueo de efectivo con desglose de billetes.

⚙️ Configuración

Personalizá todos los aspectos del sistema: colores, textos, límites, tokens, impresión. Usá el buscador para encontrar cualquier opción rápidamente.

Referencia de códigos de error del sistema. Si ves un código ERR-XXX, buscalo aquí.

CódigoDescripciónSolución
ERR-001No se pudo conectar al servidorVerificá tu conexión a internet e intentá nuevamente.
ERR-002Cliente no encontradoVerificá el nombre exacto del cliente o buscalo en Gestión de Clientes.
ERR-003Monto inválidoIngresá un número positivo sin caracteres especiales.
ERR-004CUIT inválido o no encontrado en ARCAVerificá el CUIT o consultá directamente en la web de ARCA.
ERR-005API Key de Claude no configuradaConfigurá la API Key en Configuración › API Key de Claude.
ERR-006Access Token de ARCA no configuradoConfigurá el token en Configuración › Facturación ARCA.
ERR-007Error al emitir comprobante en ARCARevisá los datos del emisor y el token. Consultá el estado de ARCA.
ERR-008Saldo excedidoEl cliente superó su límite de crédito. Podés modificarlo en Gestión de Clientes.
ERR-009Error al guardar configuraciónVerificá que los valores ingresados estén dentro del rango permitido.
ERR-010Hoja de cálculo no accesibleEjecutá inicializarSistema() desde el editor de Apps Script.
