    ❓ Centro de Ayuda - Sol & Verde V18.0

    👋 Bienvenido al Sistema Sol & Verde

    Sistema de gestión de cuentas corrientes para administrar clientes, movimientos (DEBE/HABER), y generar estadísticas en tiempo real.

    ✨ Características Principales:

    • Dashboard: Vista general de movimientos recientes y métricas
    • Visual Reasoning: Análisis de imágenes con IA Claude (tickets, facturas)
    • Visor de Clientes: Búsqueda avanzada y detalles de cuentas
    • Estadísticas: Gráficos y reportes de movimientos por período
    • Gestión de Clientes: Crear, editar y eliminar clientes
    • Configuración: Personalizar preferencias y exportar datos

    📊 Dashboard

    Función: Registrar movimientos (DEBE/HABER) y ver historial reciente.

    Cómo usar:

    1. Seleccionar cliente con búsqueda fuzzy (escribe parte del nombre)
    2. Elegir tipo: FIADO (🔴 DEBE) o PAGO (🟢 HABER)
    3. Ingresar monto y observaciones opcionales
    4. Click en "💾 Guardar Movimiento"

    🔍 Visual Reasoning

    Función: Analizar imágenes de tickets/facturas con IA Claude.

    Cómo usar:

    1. Configurar API Key de Claude en Configuración
    2. Subir imagen (JPG, PNG) o pegar desde portapapeles (Ctrl+V)
    3. IA extrae: cliente, monto, tipo (DEBE/HABER), fecha, descripción
    4. Validar y ajustar datos detectados
    5. Guardar movimientos en lote

    👥 Visor de Clientes

    Función: Ver lista completa de clientes y sus movimientos.

    Filtros disponibles:

    • 🔴 CON DEUDA: Clientes con saldo positivo (deben dinero)
    • 🟢 A FAVOR: Clientes con saldo negativo (tienen crédito)
    • 🟡 SALDO CERO: Clientes al día

    📈 Estadísticas

    Función: Gráficos de barras y métricas por período.

    Filtros:

    • Fecha desde / hasta
    • Tipo de movimiento (Todos, DEBE, HABER)
    • Cliente específico

    ⚙️ Gestión de Clientes

    Crear cliente individual: Botón "Nuevo Cliente"

    Carga masiva: Importar múltiples clientes con CSV

    Formato CSV:

    NOMBRE, TELÉFONO, EMAIL, LIMITE_CREDITO, SALDO_INICIAL, OBSERVACIONES

    🔧 Configuración

    Opciones disponibles:

    • API Key de Claude (para Visual Reasoning)
    • Tema (Claro/Oscuro)
    • Preferencias del sistema
    • Exportar datos (CSV/JSON)
    • Estadísticas del sistema

    ❓ ¿Qué significa DEBE y HABER?

    DEBE (🔴 FIADO): El cliente DEBE dinero (deuda). Aumenta el saldo.
    HABER (🟢 PAGO): El cliente PAGA o tiene crédito a favor. Disminuye el saldo.

    ❓ ¿Cómo configuro la API Key de Claude?

    1. Ir a console.anthropic.com
    2. Crear una API Key
    3. Copiar la key (empieza con "sk-ant-...")
    4. Ir a Configuración → Pegar en el campo "API Key de Claude"
    5. Click en "Guardar API Key"

    ❓ ¿Por qué no funciona Visual Reasoning?

    Causas comunes:
    • API Key no configurada o inválida
    • Imagen muy pesada (máx 5MB recomendado)
    • Formato no soportado (solo JPG/PNG)
    • Sin conexión a internet

    ❓ ¿Puedo exportar mis datos?

    Sí, en el módulo de Configuración puedes:
    • Exportar Clientes (CSV)
    • Exportar Movimientos (CSV)
    • Crear Backup Completo (JSON)

    ❓ ¿Los datos se guardan automáticamente?

    Sí, todos los cambios se guardan inmediatamente en Google Sheets. El sistema usa caché de 5 minutos para mejor performance.

    ❓ ¿Cómo elimino un cliente?

    1. Ir a Visor de Clientes
    2. Click en el cliente
    3. Click en "🗑️ Eliminar Cliente"
    ⚠️ ADVERTENCIA: Esto eliminará el cliente Y todos sus movimientos.
