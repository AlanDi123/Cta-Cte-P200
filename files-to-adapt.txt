## NOTAS IMPORTANTES

1. **doGet()**: Debe estar en main.gs para servir la Web App
2. **include()**: Helper para incluir archivos HTML parciales
3. **diagnostico_sistema.js**: Convertir a <script> e incluir al final de SistemaSolVerde.html
4. **venta-nocturna/**: Los archivos .js y .css van inline en HTML para Apps Script
5. **module.exports**: Remover de todos los archivos .gs (GAS no soporta CommonJS)
6. **Scopes OAuth**: Actualizar appsscript.json con todos los scopes necesarios

## SCOPES NECESARIOS (appsscript.json)

```json
{
  "oauthScopes": [
    "https://www.googleapis.com/auth/spreadsheets",
    "https://www.googleapis.com/auth/script.external_request",
    "https://www.googleapis.com/auth/drive",
    "https://www.googleapis.com/auth/script.scriptapp",
    "https://www.googleapis.com/auth/userinfo.email"
  ]
}
```

## PASOS DE MIGRACIÓN

1. Crear nuevo proyecto en script.google.com
2. Copiar cada archivo .gs como archivo .gs
3. Copiar SistemaSolVerde.html como archivo .html
4. Pegar contenido de diagnostico_sistema.js dentro de <script> en SistemaSolVerde.html
5. Actualizar appsscript.json (Mostrar manifiesto)
6. Guardar y probar con inicializarSistema()
7. Deploy como Web App
